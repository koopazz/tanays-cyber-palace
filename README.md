# ~*~ TANAY'S CYBER PALACE ~*~

> *Best viewed at 800x600 on Netscape Navigator 4.0 with 3D glasses and a dream.*

A love letter to the unhinged glory of early-2000s personal websites. This is not a portfolio. This is a **vibe**. Built with raw HTML, CSS, and JavaScript -- no frameworks, no bundlers, no dignity.

## What Is This

This is [Tanay Mishra's](https://www.linkedin.com/in/tanay-mishra-ai-automation/) personal site, designed to look like it was built in 2003 on GeoCities by someone who just discovered the `<marquee>` tag and couldn't be stopped.

**Features include:**

- Matrix rain background (because *The Matrix* just came out and it's still cool)
- Sparkle cursor trail that follows your mouse everywhere
- Fake Windows XP popups that try to sell you RAM
- A MySpace-style profile layout with Top 8 Friends (they're all bots)
- Dancing fire text, glitch effects, and blinking "UNDER CONSTRUCTION" banner
- A fake MIDI player with equalizer bars
- Emoji snowfall and floating rockets
- Eyes that follow your cursor around the page
- A guestbook that is powered by IMAGINATION
- Konami Code easter egg (go on, try it: up up down down left right left right B A)
- Right-click protection via *Web 1.0 DRM*
- A fake visitor counter that just makes numbers up

**Playable mini-games:**

- **Snake: Automation Edition** -- classic snake with neon aesthetics and food emojis
- **Whack-A-Bug: QA Simulator** -- 30 seconds of bug-squashing chaos
- **Cookie Clicker: Revenue Edition** -- click cookie, buy upgrades, achieve peak productivity

## Project Structure

```
pages/tanay.html      -- The main (only) page
css/style.css         -- Shared retro stylesheet (marquees, popups, neon glow, etc.)
css/tanay-custom.css  -- Tanay-specific overrides (fire text, timeline, avatar pulse)
js/effects.js         -- Sparkles, matrix rain, popups, Konami code, eye tracking
js/games.js           -- Snake, Whack-A-Bug, Cookie Clicker
assets/tanay.jpg      -- The man himself
```

## Run It

No build step. No `npm install`. Just open the file:

```bash
open pages/tanay.html
```

Or drag it into your browser like it's 2003.

## Fork It / Use It

Want your own Cyber Palace? Go for it:

1. Fork this repo
2. Replace the content in `pages/tanay.html` with your own info
3. Swap out `assets/tanay.jpg` with your face (or don't, Tanay's face is great)
4. Edit the CSS variables in `css/style.css` to change the neon color scheme
5. Deploy anywhere that serves static files (GitHub Pages, Netlify, a Raspberry Pi taped to your wall)

No attribution required, but if you want to shout out the original Cyber Palace, that's cool too.
